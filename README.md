## BlazorWebKitGtk
Build a Linux desktop app using Blazor.

## Credit
This project is based on the original **BlazorWebView** repository by **JinShil**.  
Original source: https://github.com/JinShil/BlazorWebView

## About This Fork
The original BlazorWebView project provides a Blazor Hybrid template bundled with **Bootstrap**.

This project expands on that idea by offering additional templates:

- **Minimal** — No CSS framework includes, comes with a clean setup with zero UI framework installed. 
- **Bootstrap** — includes Bootstrap version 5.


## Usage
```
git clone https://github.com/s-phal/BlazorWebKitGtk.git
cd BlazorWebKitGtk/WebKitGtk.TemplateName
dotnet run
```
